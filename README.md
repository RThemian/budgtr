# Budgtr App

## A simple budgeting app for the command line

### Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the app

### Usage

1. See a list of my income and expenses
2. Add / create a new income or expense
3. Show me one income or expense item on click

### Technical Requirements

1. Must be able to run without syntax errors
2. Must have index, show, create, new routes, using REST
3. Must have basic MVC structure
